### 工具库介绍:
#### 1.autocode
代码生成器,通过freemarker模版引擎，使用数据库表及字段信息生成:实体、spring的service、controller、mybatis的mapper.java和mapper.xml<br>
#### 2.base
包含一些基础工具类
#### 3.ddos
通过socket模拟http请求,测试服务器性能
#### 4.gson
Gson的一些序列化配置工具类
#### 5.poi
通过org.apache.poi将excel转换为json或者定义好的实体对象
#### 6.shell
java终端
#### 7.spring
基于spring的一些工具类
#### 8.tree
可展开树状结构数据
#### 9.weixin
微信api签名工具类
#### 10.zip
压缩包解压工具类
